# XTransfer 获取付款链接 — 详细操作步骤

根据 XTransfer 的实际后台功能，以下是完整操作流程。

---

## 整体说明

XTransfer 生成付款链接有 **3 种方式**，你根据实际情况选一个：

| 方式 | 适用客户 | 操作 |
|------|---------|------|
| **随心收**（推荐） | 全球客户，支持信用卡 | 后台点几下生成链接 |
| **账单收款** | 特定国家（巴西、墨西哥等） | 创建账单生成链接 |
| **全球收款账户** | 所有国家（银行转账） | 给你一个银行账号，发给客户汇款 |

**如果你的客户以美国/英国为主，优先选「随心收」或「全球收款账户」**。

---

## 步骤一：登录 XTransfer 后台

1. 打开 https://www.xtransfer.com
2. 用你的企业账号登录
3. 进入主界面

---

## 步骤二：选择你的收款方式

### 方式 A：随心收（推荐 — 最像自动支付）

**适合**：想让客户点击链接直接付款，体验最好。

**操作路径**：

```
登录后台 → 「收款」 → 「随心收」 → 「创建收款链接」
```

**填写内容**：
| 字段 | 填写 |
|------|------|
| 收款金额 | 可选（你可以在代码里固定价格，也可以手动填） |
| 币种 | USD |
| 商品描述 | "Enzyme Skincare Products" |
| 订单号 | 可选（可以后期在代码里拼接） |

**生成后得到一个链接**：
```
https://pay.xtransfer.com/pay/xxxxx
```

> ⚠️ 如果 XTransfer 认证还没下来，可能是测试链接。用真实客户之前的测试订单试一下。

### 方式 B：账单收款

**适合**：面向巴西、墨西哥、非洲等特定国家的客户。

**操作路径**：

```
登录后台 → 「收款」 → 「账单收款」 → 「创建账单」
```

填写买家信息、金额、币种 → 提交后获得一个付款链接。

### 方式 C：全球收款账户（纯银行转账）

**适合**：客户习惯银行转账，不需要在线支付体验。

**操作路径**：

```
登录后台 → 「收款」 → 「全球收款账户」
```

你会得到类似这样的信息：

```
Bank: Chase Bank NA
Account Name: YOUR COMPANY LLC
Account Number: 1234567890
Routing Number: 021000021
SWIFT Code: CHASUS33
```

把这些信息放在你网站的付款页面上，客户自己转账。

---

## 步骤三：把链接粘贴到代码里

### 如果是「随心收」或「账单收款」—— 固定链接方式

你生成一次链接，把它配置到环境变量里：

**找到文件**：`.env.local`（或 Vercel 的环境变量设置页面）

**添加**：
```
NEXT_PUBLIC_XTRANSFER_URL=https://pay.xtransfer.com/pay/xxxxx
```

代码会自动在后面拼接 `?order=ES-20260622-0001`，变成：
```
https://pay.xtransfer.com/pay/xxxxx?order=ES-20260622-0001
```

这样客户付款时备注里会自动带上订单号，你方便对账。

### 如果是「全球收款账户」—— 展示银行信息方式

通知 Codex 改方案：不需要 `NEXT_PUBLIC_XTRANSFER_URL`，而是在 Thank You 页直接显示银行账户信息。

---

## 步骤四：测试

1. 在你的网站上下个单
2. 确认跳转到 XTransfer 付款页
3. 看付款页上是否显示了订单号
4. 小额转一笔账测试
5. 钱到账后去 Admin 后台点「Confirm Payment Received」

## 常见问题

### Q: 我还没有 XTransfer 账号？
注册 → 企业认证（营业执照）→ 3-5 个工作日审核 → 开通成功。

### Q: 随心收开通不了？
随心收需要 XTransfer 开通信用卡收款资质。如果还没开：
- 用「全球收款账户」方案（给客户银行账号）
- 或者先让 Codex 把 Thank You 页改成显示银行信息模式

### Q: 美国/英国客户能通过 XTransfer 付款吗？
- **随心收**：可以（支持信用卡）
- **全球收款账户**：可以（银行转账，1-3 天到账）
- **账单收款**：目前不支持美/英

### Q: 链接是固定的还是每次生成？
**推荐固定链接方式**（最简单）：后台生成一次，一直用。Codex 在链接后面加 `?order=订单号`，你在 XTransfer 后台看备注就知道是哪一笔。
